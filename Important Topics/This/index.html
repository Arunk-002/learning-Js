• let and const: New ways to declare variables with block scope.
  • Arrow Functions: A shorter syntax for writing function expressions.
  • Template Literals: Enhanced string literals allowing embedded expressions.
  • Destructuring Assignment: A syntax for extracting values from arrays or properties from objects into distinct variables.
  • Default Parameters: Allows function parameters to have default values.
  • Spread Operators: The `...` operator for spreading elements or collecting them into an array.
  • Classes: A new syntax for creating objects and dealing with inheritance.
  • Modules: A way to organize and reuse code by exporting and importing functions, objects, or primitives.
  • Promises: A new way to handle asynchronous operations.
  • Symbol: A new primitive data type for creating unique identifiers.
  • Enhanced Object Literals: Simplified syntax for creating objects.
  • for of Loop: A new loop for iterating over iterable objects like arrays.
